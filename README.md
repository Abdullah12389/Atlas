# 🌍 Global Travel & Tour Dashboard

A highly performant, responsive country telemetry dashboard built with **React**, **TypeScript**, and **Tailwind CSS v4**, using **Vite** as the fast build pipeline. The application maps global geopolitical statistics, real-time currency matrix conversions, and utilizes a graph traversal engine to calculate overland transit border paths between nations.

## 🚀 How to Run

Follow these steps to spin up the project on a fresh machine:

### 1. Prerequisites
Ensure you have **Node.js** (v18.x or higher) and **npm** (v9.x or higher) installed.

### 2. Extract and Navigate
Open your terminal (PowerShell, Bash, or Zsh) and move into the project directory:
```bash
cd travelandtour

npm install

npm run dev
